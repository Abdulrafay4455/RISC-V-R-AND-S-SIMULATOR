# instructions.txt
# Sample RISC-V instructions for our simulator
#
# NOTE: All registers (x0-x31) start at 0. This project only has
# R-Type and S-Type instructions, there is no "li" or "addi" to load
# a starting value into a register. So if you want add/sub/etc to
# produce interesting (non-zero) results, use the "Set Register" box
# in the GUI first to set values like x1, x2, etc. before running.
#
# Format for R-Type:   op  rd  rs1  rs2
# Format for S-Type:   op  rs2  offset  rs1   (means mem[rs1+offset] = rs2)

add x17 x8 x9

or x20 x12 x13
